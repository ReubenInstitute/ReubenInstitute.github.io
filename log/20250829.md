### Investigations and discoveries

#### Parshya markers

Only two types exist: `{פ}` (open portion) and `{ס}` (closed portion).  Seven verses contained both markers in the same verse—complex but identifiable.  The decision: remove the markers from the text, but preserve the ability to record their positions for future reference if needed.

#### Footnote system

A total of 35 footnotes were found across the entire Bible, following two patterns: complete blocks and Hebrew‑letter superscripts.  Both are Sefaria editorial content—not part of the Masoretic tradition—and were marked for complete removal.

#### Kri/Ktiv (Qere/Ktiv) processing

Three fundamental types were discovered: Qere only, Ktiv only, and both.  The reading tradition (Qere) takes priority—it’s extracted as the primary text, with the written form (Ktiv) as fallback when no Qere exists.  All brackets and parentheses that mark these alternatives in Sefaria’s source are removed from the final output.

#### HTML structure analysis

Every HTML tag in the Sefaria source was catalogued and assigned a fate:
* `<sup>`, `<i>` — footnote markers: **remove**.
* `<br>` — visual formatting: **remove**.
* `<span class="mam-kq*">` — Kri/Ktiv containers: **process content, then remove**.
* `<b>`, `<small>`, `<big>` — formatting: **remove tags, keep content**.

The rule: process specific patterns first, then remove all remaining HTML tags.

#### Character encoding

HTML entities like `&nbsp;` and `&thinsp;` were converted to their actual characters after tag removal.  The legitimate Masoretic character “Combining Grapheme Joiner” was preserved—it’s part of the sacred text, not editorial noise.
