2. The Breakthrough: The Three-Layer Workflow The core of our work was establishing a robust,three-layer workflow to maximize character consistency across all generated images:

· Layer 1: Master Face Reference: We started by generating tight, detailed close-up portraits on a plain background to "lock in" each character's facial features. This prevented Midjourney from inconsistently regenerating faces across different images.
· Layer 2: Base Reference Images: Using the Master Face as an image prompt, we then generated full-body shots. The key here was that the prompt only described the body, clothing, and posture, relying on the image input for the face.
· Layer 3: Pose-Specific Images: The final layer will use both the Master Face (for identity) and a Base Reference (for clothing/style) as dual image prompts. The text prompt will then only need to describe the new pose or action.

3. Iterative Refinement of Prompts: We refined the prompts through several important iterations:

· Removing Modern Triggers: We identified and replaced terms like "rabbi," "Torah scholar," and "studio shot" that were confusing the AI with modern connotations. We switched to historically grounded terms like "1st century Judea," "learned teacher," and "sage."
· Enforcing Consistency: We added explicit commands like looking directly at the viewer and plain solid background, no environment to all prompts.
· Specifying for Consistency: For the body shots, we added highly specific details on clothing color, material, and condition (e.g., undyed wool tunic (beige-gray), frayed linen tichel (off-white)) to ensure every generation would be consistent.

· Diagnosis: We correctly identified that certain artistic terms like centered composition were triggering the unwanted cropping.
· Solution: We reformulated the prompts to use technically explicit, non-artistic language like full-length portrait, head-to-toe view, and entire body visible from head to feet, combined with strong negative prompts (--no close-up crop cropped portrait upper body medium shot).
