
# 14 Oct 2025 – DeepSeek 3 – Video.py base classes, psalm/parashah episode generation

_Media_

The video generation infrastructure took shape as a proper, reusable system.  `Video.py` was created with base classes that define the shared pipeline; `PsalmHebrewVideo` was implemented with complete functionality for Psalms; and `ParashahEpisodeVideo` was built for Parashah episodes, with the full FFmpeg pipeline set up to combine images, audio, and overlays.

### Data structures and integration

The video system integrates directly with the existing Bible modules — `Bible.py`, `BibleParashot.py`, and `BiblePsalms.py`.  Video generation is connected to the Parashah episode structure, and the image generation pipeline uses `EpisodeOverlay` for title screens and `ParashahSlide` for verse slides.

### Key features

* Fade effects — in and out transitions for a professional look.
* Background options — the classic fog effect or a solid colour, toggled per video.
* Landscape and portrait orientation support — both formats share the same generation logic.
* Audio synchronisation — verse timing data drives slide durations, so text appears exactly when it’s spoken.
* Image scaling and formatting — slides are composited correctly regardless of aspect ratio.
