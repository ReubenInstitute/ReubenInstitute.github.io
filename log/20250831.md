# 31 Aug 2025 – DeepSeek 3 – Character consistency: three‑layer Midjourney workflow

_Media_

We designed a three‑layer image‑generation pipeline that forces Midjourney to keep a character’s appearance consistent across many different poses and scenes.  The core problem was that Midjourney normally treats every prompt independently, so a character’s face, clothing, and build drift from image to image.  The three‑layer workflow breaks the problem into isolated, reusable steps.

### Layer 1 – Master Face Reference

A very tight, detailed close‑up portrait of the character’s face on a completely plain background.  The prompt focuses exclusively on immutable facial features (eye shape, nose, jawline, hairline) without any environment, clothing, or action.  This “locks in” the face so it can be reused as an image prompt in later layers.

### Layer 2 – Base Reference Images

A full‑body shot that uses the Master Face as an image prompt.  The text prompt *only* describes the body, clothing, and posture – no facial details – and the background remains plain.  This produces a consistent “reference doll” that matches the locked‑in face.

### Layer 3 – Pose‑Specific Images

Uses *both* the Master Face and the Base Reference as dual image prompts.  The text prompt describes only the new pose, action, or camera angle.  Because the identity and clothing/style are carried by the image prompts, the character remains consistent while the pose changes.

### Iterative prompt refinement

We removed modern triggers (“rabbi,” “Torah scholar,” “studio shot”) and replaced them with historically grounded terms (“1st century Judea,” “learned teacher,” “sage”).  Every prompt now enforces consistency: looking directly at the viewer, plain solid background, no environment.  Clothing details were made hyper‑specific (e.g. undyed wool tunic beige‑gray, frayed linen tichel off‑white) so every generation would produce the same wardrobe.

### Troubleshooting

Midjourney kept defaulting to cropped “prison snapshot” medium shots.  The diagnosis: artistic terms like “centered composition” were triggering the unwanted crop.  The fix: technically explicit language – “full‑length portrait,” “head‑to‑toe view,” “entire body visible from head to feet” – combined with strong negative prompts (`--no close‑up crop cropped portrait upper body medium shot`).

### Final output

A complete set of Master Face prompts and Base Body prompts for Young, Mature, and Aged versions of Akiva, Rachel, and Kalba Savua, organised into a clean Markdown table, ready to generate a full suite of consistent character assets.  The key was breaking the process into isolated steps (face first, then body) and using very precise, historically accurate language.
