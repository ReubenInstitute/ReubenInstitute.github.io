# 12 Oct 2025 – DeepSeek 3 – Overlay base class and 7 specialised renderers

_Media_

The image generation system had grown into a tangled mess — a monolithic `Image.py` with mixed responsibilities, global variables, Java‑style patterns, and no clear separation between content types.  We tore it down to the studs and rebuilt it as a clean, minimalist hierarchy.

The new `Image.py` is built around an `Overlay` base class with smart dimension handling, seven specialised renderers (four for titles, three for slide types), and a one‑method API: `save(landscape=False)`.  No factory methods, no abstract base classes, no ceremony.

### Boolean indexing

The dimension system is a single line of Zen:

```python
@property
def width(self):
    return self.SIZE[self.landscape]

@property
def height(self):
    return self.SIZE[not self.landscape]  # Zen Master style
```

Landscape and portrait swap with a boolean flip.  Every overlay inherits this automatically; no subclass ever calculates dimensions manually.

### Separation of concerns

The architecture enforces a clean line: **Layout = data structure** (what to show), **Overlay = rendering engine** (how to show it).  A paragraph knows which verses it contains; an overlay knows how to draw them.  Neither invades the other’s territory.

### Consistent patterns

Every overlay shares the same simple API.  Dimension handling is automatic.  File naming follows the content type — `psalms/`, `parashot/`, `articles/` — without any special‑case logic.

### File structure

```
Image.py
├── Image (utility class — empty for now)
├── Overlay (base with dimension magic)
├── 4 Title Overlay classes
└── 3 Slide Overlay classes
```

### The wins

We rejected Java‑style abstractions and over‑engineering at every turn.  The code embraces Python’s minimalist nature.  Every character justifies its existence.  No abstraction exists without a clear, immediate purpose.  The result is production‑ready, extensible to new content types, and a perfect foundation for the video generation phase that follows.
