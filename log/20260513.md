### The product matrix – `music` and `graphics`

When the outer session resumed, we faced the remaining naming debt.  The old `raw`/`bare`/`enhanced` terminology was finally, irrevocably killed.  Products are now defined by two honest boolean flags controlling what is *added* to the base Shmuelov narration: `music` (Suno background track) and `graphics` (custom Midjourney backgrounds).  The public labels are Library, Score, and Cinematic; the code speaks only of `music=False/True` and `graphics=False/True`.  No genre‑labels, no abbreviations, no residual ambiguity.

The final output hierarchy is governed entirely by thirteen `Path` constants in `Media.py`.  `PsalmAudio`, `PsalmVideo`, and all studio routes now import those constants and construct filenames cleanly; not a single `os.path.join` or hard‑coded folder string remains anywhere in the production modules.  The class constructors were updated to accept the boolean flags directly, and deprecated parameters such as `raw` and `cinematic` (in code) were removed.
