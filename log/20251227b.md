# 27 Dec 2025 – DeepSeek 3 – Audio system refactoring

_Studio, Media_

We separated audio generation from video generation, creating a reusable, modular audio system that can produce narration, music‑backed tracks, and telephony‑ready files for any content type.

### New class hierarchy

The old, monolithic audio logic was replaced with a clean inheritance tree:

``
BaseAudio (abstract base)
├── PsalmAudio (implements for Psalms)
└── [Planned: ParashahAudio for Torah portions]
``

Every audio object now has consistent properties: `segments` for individual verse clips with fades, `music` for an optional background music stream, `stream` for clean narration, `enhanced_stream` for narration mixed with music, and `duration` for total length.  File naming is handled by a `basename` property that guarantees consistency across all output formats.

### Export functions

Four export methods cover every distribution channel:
* `export_mp3()` — enhanced mix with narration and music.
* `export_clean_mp3()` — narration only.
* `export_sln()` — enhanced mix in telephony format for VoiceLine.
* `export_clean_sln()` — narration only for VoiceLine.

### Studio integration

A new route `/export-audio/psalm/<number>` was added, along with an “Export Audio Files” button on each psalm page.  One click generates and downloads all four formats, with an automatic redirect back to the psalm page when the export is complete.
