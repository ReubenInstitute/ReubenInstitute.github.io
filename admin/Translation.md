# Translation

## Modern Hebrew

*(Complementary to source texts, not standalone)*  
The structure must be kept exactly as in the original verses.

1. **No Concatenated Possessives**  
   Replace bound (possesive,etc) forms with analytic constructions.  
   For example:  
   - "פִּרְיוֹ" → **"הַפְּרִי שֶׁלּוֹ"**  
   - "בְּתוֹרָתוֹ" → **"בַּתּוֹרָה שֶׁלּוֹ"**  
   - "עָלָיו" → **"הֶעָלִים שֶׁלּוֹ"**  
   - "אדוני" → **"אדון שלי"**  
   - "אלוהי" → **"אלוה שלי"**  
   - סובבוני - סובבו אותי  
   - יבעתוני - יבעתו אותי  

2. **Simple Prepositions & Comparisons**  
   Modernize poetic/archaic relational terms.  
   For example:  
   - "כְּעֵץ" → **"כְּמוֹ עֵץ"**  
   - "מן הים" → **"מהים"**  

3. **Modern Vocabulary Only**  
   Substitute archaic terms with contemporary equivalents, except if the archaic term has the exact same meaning in modern Hebrew.  
   For example:  
   - "עֵת" → **"זְמַן"**  
   - "כתמול שלשלום" → **"כמו אתמול שלשלום"**  

4. **Short Parenthetical Clarifications**  
   Add ≤5-word explanations for ambiguous terms.  
   For example:  
   - "לֵצִים" → **"לֵצִים (אֲנָשִׁים מְזַלְזְלִים)"**  
   - "יָקֻמוּ" → **"יַעַמְדוּ (לֹא יִשָּׂאוּ אֶת הַדִּין)"**  

5. **Natural Verb Conjugations**  
   Update verb forms to modern usage.  
   For example:  
   - "תֹּאכֵל" → **"תֹּאכַל"**  
   - "תֹּאבֵד" → **"תֹּאֲבַד"**  
   - "ויגיע" → **"הגיע"**  
   - וייכנס → נכנס

6. **Divine Name**  
   The divine name must be kept as יהוה without diacritics, not shortened to ה or יי.
